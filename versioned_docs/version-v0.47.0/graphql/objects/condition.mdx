Condition describes the condition that must be validated by the

```graphql
type Condition {
  operator: ConditionOperator!
  value: String
}
```


### Fields

#### `operator` ([`ConditionOperator`](/docs/v0.47/graphql/enums/condition-operator))

comparator is the type of comparison to make on the value.


#### `value` ([`String`](/docs/v0.47/graphql/scalars/string))

value is used by the comparator.
