A new asset proposal change

```graphql
type NewAsset {
  name: String!
  symbol: String!
  totalSupply: String!
  decimals: Int!
  minLpStake: String!
  source: AssetSource!
}
```


### Fields

#### `name` ([`String`](/docs/v0.47/graphql/scalars/string))

The full name of the asset (e.g: Great British Pound)


#### `symbol` ([`String`](/docs/v0.47/graphql/scalars/string))

The symbol of the asset (e.g: GBP)


#### `totalSupply` ([`String`](/docs/v0.47/graphql/scalars/string))

The total supply of the market


#### `decimals` ([`Int`](/docs/v0.47/graphql/scalars/int))

The precision of the asset


#### `minLpStake` ([`String`](/docs/v0.47/graphql/scalars/string))

The min stake to become an lp for any market using this asset for settlement


#### `source` ([`AssetSource`](/docs/v0.47/graphql/unions/asset-source))

the source of the new Asset
