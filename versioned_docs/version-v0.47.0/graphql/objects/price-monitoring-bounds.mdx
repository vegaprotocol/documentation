Range of valid prices and the associated price monitoring trigger

```graphql
type PriceMonitoringBounds {
  minValidPrice: String!
  maxValidPrice: String!
  trigger: PriceMonitoringTrigger!
  referencePrice: String!
}
```


### Fields

#### `minValidPrice` ([`String`](/docs/mainnet/graphql/scalars/string))

Minimum price that isn't currently breaching the specified price monitoring trigger


#### `maxValidPrice` ([`String`](/docs/mainnet/graphql/scalars/string))

Maximum price that isn't currently breaching the specified price monitoring trigger


#### `trigger` ([`PriceMonitoringTrigger`](/docs/mainnet/graphql/objects/price-monitoring-trigger))

Price monitoring trigger associated with the bounds


#### `referencePrice` ([`String`](/docs/mainnet/graphql/scalars/string))

Reference price used to calculate the valid price range
