Subscribe to reward details data

```graphql
rewardDetails(
  assetId: ID
  party: ID
): Reward!
```


### Arguments

#### `assetId` ([`ID`](/docs/mainnet/graphql/scalars/id))

the asset to subscribe for, empty if all


#### `party` ([`ID`](/docs/mainnet/graphql/scalars/id))

the party to subscribe for, empty if all


### Type

#### [`Reward`](/docs/mainnet/graphql/objects/reward) 

Reward information for a single party
