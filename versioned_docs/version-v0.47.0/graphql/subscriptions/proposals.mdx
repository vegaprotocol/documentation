Subscribe to proposals. Leave out all arguments to receive all proposals

```graphql
proposals(
  partyId: ID
): Proposal!
```


### Arguments

#### `partyId` ([`ID`](/docs/mainnet/graphql/scalars/id))

Optional party id whose proposals are to be streamed


### Type

#### [`Proposal`](/docs/mainnet/graphql/objects/proposal)
