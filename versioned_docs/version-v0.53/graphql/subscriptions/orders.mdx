Subscribe to orders updates

```graphql
orders(
  marketId: ID
  partyId: ID
): [Order!]
```


### Arguments

#### [`marketId`](#) ([`ID`](/docs/mainnet/graphql/scalars/id))

ID of the market from which we want orders updates


#### [`partyId`](#) ([`ID`](/docs/mainnet/graphql/scalars/id))

ID of the party from which we want orders updates


### Type

#### [`Order`](/docs/mainnet/graphql/objects/order)

An order in Vega, if active it will be on the OrderBook for the market
