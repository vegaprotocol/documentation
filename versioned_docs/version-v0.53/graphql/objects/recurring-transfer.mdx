The specific details for a recurring transfer

```graphql
type RecurringTransfer {
  startEpoch: Int!
  endEpoch: Int
  factor: String!
  dispatchStrategy: DispatchStrategy
}
```


### Fields

#### [`startEpoch`](#) ([`Int!`](/docs/mainnet/graphql/scalars/int))

The epoch at which this recurring transfer will start


#### [`endEpoch`](#) ([`Int`](/docs/mainnet/graphql/scalars/int))

An optional epoch at whihc this transfer will stop


#### [`factor`](#) ([`String!`](/docs/mainnet/graphql/scalars/string))

The factor of the initial amount to be distributed


#### [`dispatchStrategy`](#) ([`DispatchStrategy`](/docs/mainnet/graphql/objects/dispatch-strategy))

An optional dispatch strategy for the recurring transfer
