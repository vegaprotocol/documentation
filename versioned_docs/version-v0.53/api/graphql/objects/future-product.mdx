### Fields

#### [`oracleSpecForTradingTermination`](#) ([`OracleSpecConfiguration!`](/mainnet/api/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine for trading termination.


#### [`oracleSpecBinding`](#) ([`OracleSpecToFutureBinding!`](/mainnet/api/graphql/objects/oracle-spec-to-future-binding))

OracleSpecToFutureBinding tells on which property oracle data should be
used as settlement price.
