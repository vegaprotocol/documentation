Subscribe to the mark price changes

```graphql
marketData(
  marketId: ID
): MarketData!
```


### Arguments

#### [`marketId`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

id of the market we want to subscribe to the market data changes


### Type

#### [`MarketData`](/mainnet/api/graphql/objects/market-data)

Live data of a Market
