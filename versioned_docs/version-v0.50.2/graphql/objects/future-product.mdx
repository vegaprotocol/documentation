No description

```graphql
type FutureProduct {
  settlementAsset: Asset!
  quoteName: String!
  oracleSpecForSettlementPrice: OracleSpecConfiguration!
  oracleSpecForTradingTermination: OracleSpecConfiguration!
  oracleSpecBinding: OracleSpecToFutureBinding!
}
```


### Fields

#### `settlementAsset` ([`Asset`](/docs/mainnet/graphql/objects/asset))

Product asset ID


#### `quoteName` ([`String`](/docs/mainnet/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -&#x003E; USD is quote)


#### `oracleSpecForSettlementPrice` ([`OracleSpecConfiguration`](/docs/mainnet/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine for settlement price.


#### `oracleSpecForTradingTermination` ([`OracleSpecConfiguration`](/docs/mainnet/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine for trading termination.


#### `oracleSpecBinding` ([`OracleSpecToFutureBinding`](/docs/mainnet/graphql/objects/oracle-spec-to-future-binding))

OracleSpecToFutureBinding tells on which property oracle data should be
used as settlement price.
