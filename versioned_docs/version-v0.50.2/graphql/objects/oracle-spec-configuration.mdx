title: OracleSpecConfiguration
---


An oracle spec describe the oracle data that an instrument wants to get from the
oracle engine.

```graphql
type OracleSpecConfiguration {
  pubKeys: [String!]
  filters: [Filter!]
}
```


### Fields

#### `pubKeys` ([`String`](/docs/mainnet/graphql/scalars/string))

pubKeys is the list of authorized public keys that signed the data for this
oracle. All the public keys in the oracle data should be contained in these
public keys.


#### `filters` ([`Filter`](/docs/mainnet/graphql/objects/filter))

filters describes which oracle data are considered of interest or not for
the product (or the risk model).
