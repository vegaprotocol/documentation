Statistics about the node

```graphql
type Statistics {
  blockHeight: String!
  blockHash: String!
  backlogLength: String!
  totalPeers: String!
  genesisTime: String!
  currentTime: String!
  upTime: String!
  vegaTime: String!
  status: String!
  txPerBlock: String!
  averageTxBytes: String!
  averageOrdersPerBlock: String!
  tradesPerSecond: String!
  ordersPerSecond: String!
  totalMarkets: String!
  totalAmendOrder: String!
  totalCancelOrder: String!
  totalCreateOrder: String!
  totalOrders: String!
  totalTrades: String!
  appVersionHash: String!
  appVersion: String!
  chainVersion: String!
  blockDuration: String!
  chainId: String!
}
```


### Fields

#### `blockHeight` ([`String`](/docs/testnet/graphql/scalars/string))

Current block number


#### `blockHash` ([`String`](/docs/testnet/graphql/scalars/string))

Current block hash


#### `backlogLength` ([`String`](/docs/testnet/graphql/scalars/string))

Number of items in the backlog


#### `totalPeers` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of peers on the vega network


#### `genesisTime` ([`String`](/docs/testnet/graphql/scalars/string))

RFC3339Nano genesis time of the chain


#### `currentTime` ([`String`](/docs/testnet/graphql/scalars/string))

RFC3339Nano current time (real)


#### `upTime` ([`String`](/docs/testnet/graphql/scalars/string))

RFC3339Nano uptime of the node


#### `vegaTime` ([`String`](/docs/testnet/graphql/scalars/string))

RFC3339Nano current time of the chain (decided through consensus)


#### `status` ([`String`](/docs/testnet/graphql/scalars/string))

Status of the vega application connection with the chain


#### `txPerBlock` ([`String`](/docs/testnet/graphql/scalars/string))

Number of transaction processed per block


#### `averageTxBytes` ([`String`](/docs/testnet/graphql/scalars/string))

Average size of the transactions


#### `averageOrdersPerBlock` ([`String`](/docs/testnet/graphql/scalars/string))

Average number of orders added per blocks


#### `tradesPerSecond` ([`String`](/docs/testnet/graphql/scalars/string))

Number of the trades per seconds


#### `ordersPerSecond` ([`String`](/docs/testnet/graphql/scalars/string))

Number of orders per seconds


#### `totalMarkets` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of markets


#### `totalAmendOrder` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of amended orders


#### `totalCancelOrder` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of cancelled orders


#### `totalCreateOrder` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of orders created


#### `totalOrders` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of orders


#### `totalTrades` ([`String`](/docs/testnet/graphql/scalars/string))

Total number of trades


#### `appVersionHash` ([`String`](/docs/testnet/graphql/scalars/string))

Version commit hash of the vega node


#### `appVersion` ([`String`](/docs/testnet/graphql/scalars/string))

Version of the vega node (semver)


#### `chainVersion` ([`String`](/docs/testnet/graphql/scalars/string))

Version of the chain (semver)


#### `blockDuration` ([`String`](/docs/testnet/graphql/scalars/string))

Duration of the last block, in nanoseconds


#### `chainId` ([`String`](/docs/testnet/graphql/scalars/string))

Current chain id
