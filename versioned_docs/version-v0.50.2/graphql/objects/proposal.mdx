No description

```graphql
type Proposal {
  id: ID
  reference: String!
  party: Party!
  state: ProposalState!
  datetime: String!
  terms: ProposalTerms!
  votes: ProposalVotes!
  rejectionReason: ProposalRejectionReason
  errorDetails: String
}
```


### Fields

#### `id` ([`ID`](/docs/mainnet/graphql/scalars/id))

Proposal ID that is filled by VEGA once proposal reaches the network


#### `reference` ([`String`](/docs/mainnet/graphql/scalars/string))

A UUID reference to aid tracking proposals on VEGA


#### `party` ([`Party`](/docs/mainnet/graphql/objects/party))

Party that prepared the proposal


#### `state` ([`ProposalState`](/docs/mainnet/graphql/enums/proposal-state))

State of the proposal


#### `datetime` ([`String`](/docs/mainnet/graphql/scalars/string))

RFC3339Nano time and date when the proposal reached Vega network


#### `terms` ([`ProposalTerms`](/docs/mainnet/graphql/objects/proposal-terms))

Terms of the proposal


#### `votes` ([`ProposalVotes`](/docs/mainnet/graphql/objects/proposal-votes))

Votes cast for this proposal


#### `rejectionReason` ([`ProposalRejectionReason`](/docs/mainnet/graphql/enums/proposal-rejection-reason))

Reason for the proposal to be rejected by the core


#### `errorDetails` ([`String`](/docs/mainnet/graphql/scalars/string))

Error details of the rejectionReason
