#### `inactiveNodes` ([`Int`](/docs/testnet/graphql/scalars/int))

#### `validatingNodes` ([`Int`](/docs/testnet/graphql/scalars/int))

#### `uptime` ([`Float`](/docs/testnet/graphql/scalars/float))
