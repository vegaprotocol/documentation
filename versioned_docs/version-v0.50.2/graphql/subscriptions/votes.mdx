Subscribe to votes, either by proposal id or party id

```graphql
votes(
  proposalId: ID
  partyId: ID
): ProposalVote!
```


### Arguments

#### `proposalId` ([`ID`](/docs/mainnet/graphql/scalars/id))

Optional proposal id which votes are to be streamed


#### `partyId` ([`ID`](/docs/mainnet/graphql/scalars/id))

Optional party id whose votes are to be streamed


### Type

#### [`ProposalVote`](/docs/mainnet/graphql/objects/proposal-vote)
