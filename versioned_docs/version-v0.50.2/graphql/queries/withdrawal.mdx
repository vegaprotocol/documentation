find a withdrawal using its id

```graphql
withdrawal(
  id: ID!
): Withdrawal
```


### Arguments

#### `id` ([`ID`](/docs/mainnet/graphql/scalars/id))

id of the withdrawal


### Type

#### [`Withdrawal`](/docs/mainnet/graphql/objects/withdrawal) 

The details of a withdrawal processed by vega
