Order versions (created via amendments if any) found by orderID

```graphql
orderVersions(
  orderId: ID!
  skip: Int
  first: Int
  last: Int
): [Order!]
```


### Arguments

#### `orderId` ([`ID`](/docs/mainnet/graphql/scalars/id))

ID for an order


#### `skip` ([`Int`](/docs/mainnet/graphql/scalars/int))

Pagination skip


#### `first` ([`Int`](/docs/mainnet/graphql/scalars/int))

Pagination first element


#### `last` ([`Int`](/docs/mainnet/graphql/scalars/int))

Pagination last element


### Type

#### [`Order`](/docs/mainnet/graphql/objects/order) 

An order in Vega, if active it will be on the OrderBook for the market
