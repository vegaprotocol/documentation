All governance proposals in the VEGA network

```graphql
proposals(
  inState: ProposalState
): [Proposal!]
```


### Arguments

#### `inState` ([`ProposalState`](/docs/mainnet/graphql/enums/proposal-state))

Returns only proposals in the specified state. Leave out to get all proposals


### Type

#### [`Proposal`](/docs/mainnet/graphql/objects/proposal)
