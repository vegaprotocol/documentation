title: Interval
---


The interval for trade candles when subscribing via VEGA graphql, default is I15M

```graphql
enum Interval {
  I1M
  I5M
  I15M
  I1H
  I6H
  I1D
}
```


### Values

#### `I1M` 

1 minute interval


#### `I5M` 

5 minute interval


#### `I15M` 

15 minute interval (default)


#### `I1H` 

1 hour interval


#### `I6H` 

6 hour interval


#### `I1D` 

1 day interval
