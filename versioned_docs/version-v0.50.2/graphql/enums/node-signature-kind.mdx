title: NodeSignatureKind
---


Represents the type signature provided by a node

```graphql
enum NodeSignatureKind {
  AssetNew
  AssetWithdrawal
}
```


### Values

#### `AssetNew` 

A signature for proposing a new asset into the network


#### `AssetWithdrawal` 

A signature for allowing a withdrawal of funds
