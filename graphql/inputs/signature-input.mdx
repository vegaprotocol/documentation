A signature to be bundled with a transaction

```graphql
type SignatureInput {
  sig: String!
  algo: String!
  version: Int!
}
```

### Fields

#### `sig` ([`String!`](/graphql/scalars/string))

The signature, base64 encoded

#### `algo` ([`String!`](/graphql/scalars/string))

The algorithm used to produice the signature

#### `version` ([`Int!`](/graphql/scalars/int))

The version of the signature
