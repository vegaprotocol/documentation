```graphql
type Vote {

  governanceTokenWeight: String!
}
```

### Fields

#### `value` ([`VoteValue!`](/graphql/enums/vote-value))

The vote value cast

#### `party` ([`Party!`](/graphql/objects/party))

The party casting the vote

#### `datetime` ([`String!`](/graphql/scalars/string))

RFC3339Nano time and date when the vote reached Vega network

#### `proposalId` ([`ID!`](/graphql/scalars/id))

The ID of the proposal this vote applies to

#### `governanceTokenBalance` ([`String!`](/graphql/scalars/string))

Total number of governance token for the party that casted the vote

#### `governanceTokenWeight` ([`String!`](/graphql/scalars/string))

The weight of this vote based on the total of governance token
