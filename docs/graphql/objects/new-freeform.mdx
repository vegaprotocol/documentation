A new freeform proposal change

```graphql
type NewFreeform {
  url: String!
  description: String!
  hash: String!
}
```


### Fields

#### `url` ([`String`](/docs/testnet/graphql/scalars/string))

The URL containing content that describes the proposal


#### `description` ([`String`](/docs/testnet/graphql/scalars/string))

A short description of what is being proposed


#### `hash` ([`String`](/docs/testnet/graphql/scalars/string))

The hash on the content of the URL
