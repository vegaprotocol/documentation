PropertyKey describes the property key contained in an oracle data.

```graphql
type PropertyKey {
  name: String
  type: PropertyKeyType!
}
```


### Fields

#### [`name`](#) ([`String`](/docs/testnet/graphql/scalars/string))

name is the name of the property.


#### [`type`](#) ([`PropertyKeyType!`](/docs/testnet/graphql/enums/property-key-type))

type is the type of the property.
