Market Depth Update is a delta to the current market depth which can be used to update the
market depth structure to keep it correct

```graphql
type MarketDepthUpdate {
  market: Market!
  buy: [PriceLevel!]
  sell: [PriceLevel!]
  sequenceNumber: String!
}
```


### Fields

#### `market` ([`Market`](/docs/testnet/graphql/objects/market))

Market id


#### `buy` ([`PriceLevel`](/docs/testnet/graphql/objects/price-level))

Buy side price levels (if available)


#### `sell` ([`PriceLevel`](/docs/testnet/graphql/objects/price-level))

Sell side price levels (if available)


#### `sequenceNumber` ([`String`](/docs/testnet/graphql/scalars/string))

Sequence number for the current snapshot of the market depth
