title: OracleData
---

An oracle data contains the data sent by an oracle

```graphql
type OracleData {
  pubKeys: [String!]
  data: [Property!]
}
```

### Fields

#### `pubKeys` ([`[String!]`](/docs/graphql/scalars/string))

pubKeys is the list of public keys that signed the data

#### `data` ([`[Property!]`](/docs/graphql/objects/property))

data contains all the properties send by an oracle
