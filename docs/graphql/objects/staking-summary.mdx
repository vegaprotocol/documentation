All staking information related to a Party.
Contains the current recognised balance by the network and
all the StakeLink/Unlink seen by the network

```graphql
type StakingSummary {
  currentStakeAvailable: String!
  linkings(
  pagination: Pagination
): StakesConnection!
}
```
