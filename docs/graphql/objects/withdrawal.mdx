The details of a withdrawal processed by Vega

```graphql
type Withdrawal {
  id: ID!
  party: Party!
  amount: String!
  asset: Asset!
  status: WithdrawalStatus!
  ref: String!
  expiry: String!
  createdTimestamp: String!
  withdrawnTimestamp: String
  txHash: String
  details: WithdrawalDetails
}
```


### Fields

#### [`id`](#) ([`ID!`](/docs/testnet/graphql/scalars/id))

The Vega internal ID of the withdrawal


#### [`party`](#) ([`Party!`](/docs/testnet/graphql/objects/party))

The Party initiating the withdrawal


#### [`amount`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The amount to be withdrawn


#### [`asset`](#) ([`Asset!`](/docs/testnet/graphql/objects/asset))

The asset to be withdrawn


#### [`status`](#) ([`WithdrawalStatus!`](/docs/testnet/graphql/enums/withdrawal-status))

The current status of the withdrawal


#### [`ref`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

A reference the foreign chain can use to refer to when processing the withdrawal


#### [`expiry`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

RFC3339Nano time until the withdrawal will be invalid


#### [`createdTimestamp`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

RFC3339Nano time at which the withdrawal was created


#### [`withdrawnTimestamp`](#) ([`String`](/docs/testnet/graphql/scalars/string))

RFC3339Nano time at which the withdrawal was finalised


#### [`txHash`](#) ([`String`](/docs/testnet/graphql/scalars/string))

#### [`details`](#) ([`WithdrawalDetails`](/docs/testnet/graphql/unions/withdrawal-details))
