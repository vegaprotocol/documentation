PriceMonitoringParameters holds together price projection horizon τ, probability level p, and auction extension duration

```graphql
type PriceMonitoringTrigger {
  horizonSecs: Int!
  probability: Float!
  auctionExtensionSecs: Int!
}
```


### Fields

#### [`horizonSecs`](#) ([`Int!`](/docs/testnet/graphql/scalars/int))

Price monitoring projection horizon τ in seconds (&#x003E; 0).


#### [`probability`](#) ([`Float!`](/docs/testnet/graphql/scalars/float))

Price monitoring probability level p. (&#x003E;0 and &#x003C; 1)


#### [`auctionExtensionSecs`](#) ([`Int!`](/docs/testnet/graphql/scalars/int))

Price monitoring auction extension duration in seconds should the price
breach its theoretical level over the specified horizon at the specified
probability level (&#x003E; 0)
