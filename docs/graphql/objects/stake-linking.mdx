A stake linking represent the intent from a party to deposit / remove stake on their account

```graphql
type StakeLinking {
  id: ID!
  type: StakeLinkingType!
  timestamp: String!
  party: Party!
  amount: String!
  status: StakeLinkingStatus!
  finalizedAt: String
  txHash: String!
}
```


### Fields

#### [`id`](#) ([`ID!`](/docs/testnet/graphql/scalars/id))




#### [`type`](#) ([`StakeLinkingType!`](/docs/testnet/graphql/enums/stake-linking-type))

Type of linking: link|unlink


#### [`timestamp`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The time at which the request happened on ethereum


#### [`party`](#) ([`Party!`](/docs/testnet/graphql/objects/party))

The party initiating the stake linking


#### [`amount`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The amount linked or unlinked


#### [`status`](#) ([`StakeLinkingStatus!`](/docs/testnet/graphql/enums/stake-linking-status))

The status of the linking


#### [`finalizedAt`](#) ([`String`](/docs/testnet/graphql/scalars/string))

The time at which the stake linking was fully processed by the Vega network, null until defined


#### [`txHash`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The transaction hash (ethereum) which initiated the link/unlink
