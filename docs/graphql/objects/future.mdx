A Future product

```graphql
type Future {
  settlementAsset: Asset!
  quoteName: String!
  oracleSpecForSettlementPrice: OracleSpec!
  oracleSpecForTradingTermination: OracleSpec!
  oracleSpecBinding: OracleSpecToFutureBinding!
  settlementPriceDecimals: Int!
}
```


### Fields

#### [`settlementAsset`](#) ([`Asset!`](/docs/testnet/graphql/objects/asset))

The name of the asset (string)


#### [`quoteName`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -&#x003E; USD is quote)


#### [`oracleSpecForSettlementPrice`](#) ([`OracleSpec!`](/docs/testnet/graphql/objects/oracle-spec))

The oracle spec describing the oracle data of interest for settlement price.


#### [`oracleSpecForTradingTermination`](#) ([`OracleSpec!`](/docs/testnet/graphql/objects/oracle-spec))

The oracle spec describing the oracle data of interest for trading termination.


#### [`oracleSpecBinding`](#) ([`OracleSpecToFutureBinding!`](/docs/testnet/graphql/objects/oracle-spec-to-future-binding))

The binding between the oracle spec and the settlement price


#### [`settlementPriceDecimals`](#) ([`Int!`](/docs/testnet/graphql/scalars/int))

The number of decimal places implied by the settlement price emitted by the settlement oracle
