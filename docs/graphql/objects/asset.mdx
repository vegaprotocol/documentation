Represents an asset in vega

```graphql
type Asset {
  id: ID!
  name: String!
  symbol: String!
  totalSupply: String!
  decimals: Int!
  quantum: String!
  source: AssetSource!
  status: AssetStatus!
  infrastructureFeeAccount: Account!
  globalRewardPoolAccount: Account
}
```


### Fields

#### [`id`](#) ([`ID!`](/docs/testnet/graphql/scalars/id))

The id of the asset


#### [`name`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The full name of the asset (e.g: Great British Pound)


#### [`symbol`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The symbol of the asset (e.g: GBP)


#### [`totalSupply`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The total supply of the market


#### [`decimals`](#) ([`Int!`](/docs/testnet/graphql/scalars/int))

The precision of the asset


#### [`quantum`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The minimum economically meaningful amount in the asset


#### [`source`](#) ([`AssetSource!`](/docs/testnet/graphql/unions/asset-source))

The origin source of the asset (e.g: an erc20 asset)


#### [`status`](#) ([`AssetStatus!`](/docs/testnet/graphql/enums/asset-status))

The status of the asset in the vega network


#### [`infrastructureFeeAccount`](#) ([`Account!`](/docs/testnet/graphql/objects/account))

The infrastructure fee account for this asset


#### [`globalRewardPoolAccount`](#) ([`Account`](/docs/testnet/graphql/objects/account))

The global reward pool account for this asset
