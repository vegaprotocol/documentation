A new asset proposal change

```graphql
type NewAsset {
  name: String!
  symbol: String!
  decimals: Int!
  minLpStake: String!
  source: AssetSource!
}
```


### Fields

#### [`name`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The full name of the asset (e.g: Great British Pound)


#### [`symbol`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The symbol of the asset (e.g: GBP)


#### [`decimals`](#) ([`Int!`](/docs/testnet/graphql/scalars/int))

The precision of the asset


#### [`minLpStake`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

The minimum stake to become a liquidity provider for any market using this asset for settlement


#### [`source`](#) ([`AssetSource!`](/docs/testnet/graphql/unions/asset-source))

The source of the new asset
