The equity like share of liquidity fee for each liquidity provider

```graphql
type LiquidityProviderFeeShare {
  party: Party!
  equityLikeShare: String!
  averageEntryValuation: String!
}
```

### Fields

#### `party` ([`Party!`](/docs/graphql/objects/party))

The liquidity provider party id

#### `equityLikeShare` ([`String!`](/docs/graphql/scalars/string))

The share own by this liquidity provider (float)

#### `averageEntryValuation` ([`String!`](/docs/graphql/scalars/string))

the average entry valuation of the liqidity provider for the market
