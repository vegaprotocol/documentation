The type of stake linking

```graphql
enum StakeLinkingType {
  Link
  Unlink
}
```


### Values

#### [`Link`](#)

The stake is being linked (deposited) to a vega stake account


#### [`Unlink`](#)

The stake is being unlined (removed) from a vega stake account
