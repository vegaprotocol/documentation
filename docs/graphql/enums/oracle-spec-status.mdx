Status describe the status of the oracle spec

```graphql
enum OracleSpecStatus {
  StatusActive
  StatusUnused
}
```


### Values

#### [`StatusActive`](#)

StatusActive describes an active oracle spec.


#### [`StatusUnused`](#)

StatusUnused describes an oracle spec that is not listening to data
anymore.
