Status describe the status of the oracle spec

```graphql
enum OracleSpecStatus {
  STATUS_ACTIVE
  STATUS_DEACTIVATED
}
```


### Values

#### [`STATUS_ACTIVE`](#)

describes an active oracle spec.


#### [`STATUS_DEACTIVATED`](#)

describes an oracle spec that is not listening to data
anymore.
