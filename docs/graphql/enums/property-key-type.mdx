Type describes the type of properties that are supported by the oracle
engine.

```graphql
enum PropertyKeyType {
  TypeEmpty
  TypeInteger
  TypeString
  TypeBoolean
  TypeDecimal
  TypeTimestamp
}
```


### Values

#### [`TypeEmpty`](#)

Any type.


#### [`TypeInteger`](#)

Integer type.


#### [`TypeString`](#)

String type.


#### [`TypeBoolean`](#)

Boolean type.


#### [`TypeDecimal`](#)

Any floating point decimal type.


#### [`TypeTimestamp`](#)

Timestamp date type.
