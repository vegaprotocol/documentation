Type describes the type of properties that are supported by the oracle
engine.

```graphql
enum PropertyKeyType {
  TYPE_EMPTY
  TYPE_INTEGER
  TYPE_STRING
  TYPE_BOOLEAN
  TYPE_DECIMAL
  TYPE_TIMESTAMP
}
```


### Values

#### [`TYPE_EMPTY`](#)

Any type.


#### [`TYPE_INTEGER`](#)

Integer type.


#### [`TYPE_STRING`](#)

String type.


#### [`TYPE_BOOLEAN`](#)

Boolean type.


#### [`TYPE_DECIMAL`](#)

Any floating point decimal type.


#### [`TYPE_TIMESTAMP`](#)

Timestamp date type.
