Directs the executor to skip this field or fragment when the `if` argument is true.

```graphql
directive @skip(
  if: Boolean!
)
```


### Arguments

#### [`if`](#) ([`Boolean!`](/docs/testnet/graphql/scalars/boolean))

Skipped when true.
