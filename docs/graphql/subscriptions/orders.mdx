Subscribe to orders updates

```graphql
orders(
  marketId: ID
  partyId: ID
): [OrderUpdate!]
```


### Arguments

#### [`marketId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market from which you want orders updates


#### [`partyId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the party from which you want orders updates


### Type

#### [`OrderUpdate`](/docs/testnet/graphql/objects/order-update)

An order update in Vega, if active it will be on the order book for the market
