Subscribe to the accounts updates

```graphql
accounts(
  marketId: ID
  partyId: ID
  assetId: ID
  type: AccountType
): [AccountUpdate!]!
```


### Arguments

#### [`marketId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market from which to receive accounts updates for


#### [`partyId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the party from which to receive accounts updates for


#### [`assetId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

Asset code


#### [`type`](#) ([`AccountType`](/docs/testnet/graphql/enums/account-type))

Type of the account


### Type

#### [`AccountUpdate`](/docs/testnet/graphql/objects/account-update)

An account record used for subscriptions
