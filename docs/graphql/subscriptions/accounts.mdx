Subscribe to the accounts updates

```graphql
accounts(
  marketId: ID
  partyId: ID
  asset: String
  type: AccountType
): Account!
```


### Arguments

#### `marketId` ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market from which we want accounts updates


#### `partyId` ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the party from which we want accounts updates


#### `asset` ([`String`](/docs/testnet/graphql/scalars/string))

Asset code


#### `type` ([`AccountType`](/docs/testnet/graphql/enums/account-type))

Type of the account


### Type

#### [`Account`](/docs/testnet/graphql/objects/account) 

An account record
