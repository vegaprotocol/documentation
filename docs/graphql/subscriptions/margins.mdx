Subscribe to the margin changes

```graphql
margins(
  partyId: ID!
  marketId: ID
): MarginLevelsUpdate!
```


### Arguments

#### [`partyId`](#) ([`ID!`](/docs/testnet/graphql/scalars/id))

ID of the party you want to subscribe to for margin updates


#### [`marketId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market you want to listen to for margin updates (nil if you want updates for all markets)


### Type

#### [`MarginLevelsUpdate`](/docs/testnet/graphql/objects/margin-levels-update)

Margins for a given a party
