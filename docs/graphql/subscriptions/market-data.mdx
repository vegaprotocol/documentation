Subscribe to the mark price changes

```graphql
marketData(
  marketId: ID
): MarketData!
```


### Arguments

#### [`marketId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market you want to subscribe to the market data changes


### Type

#### [`MarketData`](/docs/testnet/graphql/objects/market-data)

Live data of a Market
