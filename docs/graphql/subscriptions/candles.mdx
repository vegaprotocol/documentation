Subscribe to the candles updates

```graphql
candles(
  marketId: ID!
  interval: Interval!
): Candle!
```


### Arguments

#### `marketId` ([`ID`](/docs/testnet/graphql/scalars/id))

ID of the market we want to listen candles for


#### `interval` ([`Interval`](/docs/testnet/graphql/enums/interval))

Interval of the candles we want to listen for


### Type

#### [`Candle`](/docs/testnet/graphql/objects/candle) 

Candle stick representation of trading
