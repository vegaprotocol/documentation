Subscribe to the mark price changes

```graphql
marketsData(
  marketIds: [ID!]!
): [ObservableMarketData!]!
```


### Arguments

#### [`marketIds`](#) ([`[ID!]!`](/docs/testnet/graphql/scalars/id))

ID of the market for which you want to subscribe to the market data changes


### Type

#### [`ObservableMarketData`](/docs/testnet/graphql/objects/observable-market-data)

Live data of a Market
