Subscribe to delegation data

```graphql
delegations(
  partyId: ID
  nodeId: ID
): Delegation!
```


### Arguments

#### [`partyId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

the party to subscribe for, empty if all


#### [`nodeId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

the node to subscribe for, empty if all


### Type

#### [`Delegation`](/docs/testnet/graphql/objects/delegation)
