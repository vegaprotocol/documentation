Subscribe to reward details data

```graphql
rewards(
  assetId: ID
  party: ID
): Reward!
```


### Arguments

#### [`assetId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

the asset to subscribe for, empty if all


#### [`party`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

the party to subscribe for, empty if all


### Type

#### [`Reward`](/docs/testnet/graphql/objects/reward)

Reward information for a single party
