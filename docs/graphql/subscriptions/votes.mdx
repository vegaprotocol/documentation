Subscribe to votes, either by proposal ID or party ID

```graphql
votes(
  proposalId: ID
  partyId: ID
): ProposalVote!
```


### Arguments

#### [`proposalId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

Optional proposal ID which votes are to be streamed


#### [`partyId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

Optional party ID whose votes are to be streamed


### Type

#### [`ProposalVote`](/docs/testnet/graphql/objects/proposal-vote)
