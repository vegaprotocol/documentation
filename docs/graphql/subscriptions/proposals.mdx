Subscribe to proposals. Leave out all arguments to receive all proposals

```graphql
proposals(
  partyId: ID
): Proposal!
```


### Arguments

#### [`partyId`](#) ([`ID`](/docs/testnet/graphql/scalars/id))

Optional party ID whose proposals are to be streamed

#### [`Proposal`](/docs/testnet/graphql/objects/proposal)
