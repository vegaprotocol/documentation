get a list of all transfers for a public key

```graphql
transfersConnection(
  partyId: String
  direction: TransferDirection
  pagination: Pagination

```


### Arguments

#### [`partyId`](#) ([`String`](/docs/testnet/graphql/scalars/string))

the public key to look for


#### [`direction`](#) ([`TransferDirection`](/docs/testnet/graphql/enums/transfer-direction))

direction of the transfer with respect to the public key


#### [`pagination`](#) ([`Pagination`](/docs/testnet/graphql/inputs/pagination))

Pagination information
