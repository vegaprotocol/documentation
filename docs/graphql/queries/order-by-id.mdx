An order in the Vega network found by orderID

```graphql
orderByID(
  id: ID!
  version: Int
): Order!
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>orderByID.<b>id</b></code>](#)<Bullet />[`ID!`](/docs/testnet/graphql/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> ID for an order
> 

#### [<code style={{ fontWeight: 'normal' }}>orderByID.<b>version</b></code>](#)<Bullet />[`Int`](/docs/testnet/graphql/scalars/int) <Badge class="secondary" text="scalar"/>
> version of the order (omitted or 0 for most recent; 1 for original; 2 for first amendment, etc)
> 

### Type

#### [`Order`](/docs/testnet/graphql/objects/order) <Badge class="secondary" text="object"/>
> An order in Vega, if active it will be on the order book for the market
>
