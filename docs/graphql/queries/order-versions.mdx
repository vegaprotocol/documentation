Order versions (created via amendments if any) found by orderID

```graphql
orderVersions(
  orderId: ID!
  skip: Int
  first: Int
  last: Int
): [Order!]
```


### Arguments

#### [`orderId`](#) ([`ID!`](/docs/testnet/graphql/scalars/id))

ID for an order


#### [`skip`](#) ([`Int`](/docs/testnet/graphql/scalars/int))

Pagination skip


#### [`first`](#) ([`Int`](/docs/testnet/graphql/scalars/int))

Pagination first element


#### [`last`](#) ([`Int`](/docs/testnet/graphql/scalars/int))

Pagination last element


### Type

#### [`Order`](/docs/testnet/graphql/objects/order)

An order in Vega, if active it will be on the order book for the market
