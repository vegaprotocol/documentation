All oracle data for a given oracle spec ID

```graphql
oracleDataBySpec(
  oracleSpecID: String!
  pagination: OffsetPagination
): [OracleData!]
```


### Arguments

#### [`oracleSpecID`](#) ([`String!`](/docs/testnet/graphql/scalars/string))

ID for an oracle spec


#### [`pagination`](#) ([`OffsetPagination`](/docs/testnet/graphql/inputs/offset-pagination))

Pagination


### Type

#### [`OracleData`](/docs/testnet/graphql/objects/oracle-data)

An oracle data contains the data sent by an oracle
