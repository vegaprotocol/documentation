All registered oracle specs

```graphql
oracleSpecs(
  pagination: OffsetPagination
): [OracleSpec!]
```


### Arguments

#### `pagination` ([`OffsetPagination`](/docs/testnet/graphql/inputs/offset-pagination))

Pagination


### Type

#### [`OracleSpec`](/docs/testnet/graphql/objects/oracle-spec) 

An oracle spec describe the oracle data that a product (or a risk model)
wants to get from the oracle engine.
