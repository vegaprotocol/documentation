All governance proposals in the Vega network

```graphql
proposals(
  inState: ProposalState
): [Proposal!]
```


### Arguments

#### [`inState`](#) ([`ProposalState`](/docs/testnet/graphql/enums/proposal-state))

Returns only proposals in the specified state. Leave out to get all proposals


### Type

#### [`Proposal`](/docs/testnet/graphql/objects/proposal)
