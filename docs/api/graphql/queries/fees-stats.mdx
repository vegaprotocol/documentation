Get fees statistics

```graphql
feesStats(
  marketId: ID
  assetId: ID
  epoch: Int
  partyId: ID
): FeesStats
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>feesStats.<b>marketId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optional market ID to filter for. If omitted, asset ID must be provided.
> 

#### [<code style={{ fontWeight: 'normal' }}>feesStats.<b>assetId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optional asset ID to filter for. If omitted, market ID must be provided.
> 

#### [<code style={{ fontWeight: 'normal' }}>feesStats.<b>epoch</b></code>](#)<Bullet />[`Int`](/testnet/api/graphql/scalars/int) <Badge class="secondary" text="scalar"/>
> Optional epoch to filter for. If omitted, the most recent epoch's data is returned.
> 

#### [<code style={{ fontWeight: 'normal' }}>feesStats.<b>partyId</b></code>](#)<Bullet />[`ID`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> Optional party ID to filter for
> 

### Type

#### [`FeesStats`](/testnet/api/graphql/objects/fees-stats) <Badge class="secondary" text="object"/>
> Fees that have been applied on a specific market/asset up to the given epoch.
