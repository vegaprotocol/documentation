Referral program information reported by data node with additional endedAt timestamp.

```graphql
type CurrentReferralProgram {
  id: ID!
  version: Int!
  benefitTiers: [BenefitTier!]!
  endOfProgramTimestamp: Timestamp!
  windowLength: Int!
  stakingTiers: [StakingTier!]!
  endedAt: Timestamp
}
```

> 

#### [<code style={{ fontWeight: 'normal' }}>CurrentReferralProgram.<b>windowLength</b></code>](#)<Bullet />[`Int!`](/testnet/api/graphql/scalars/int) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

> 

#### [<code style={{ fontWeight: 'normal' }}>CurrentReferralProgram.<b>stakingTiers</b></code>](#)<Bullet />[`[StakingTier!]!`](/testnet/api/graphql/objects/staking-tier) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="object"/>
> Defined staking tiers in increasing order. First element will give Tier 1,
second element will give Tier 2, and so on. Determines the level of

#### [<code style={{ fontWeight: 'normal' }}>CurrentReferralProgram.<b>endedAt</b></code>](#)<Bullet />[`Timestamp`](/testnet/api/graphql/scalars/timestamp) <Badge class="secondary" text="scalar"/>
