An data source specification describes the data source data that a product (or a risk model)
wants to get from the oracle engine.

```graphql
type DataSourceSpec {
  id: ID!
  createdAt: Timestamp!
  updatedAt: Timestamp
  data: DataSourceDefinition!
  status: DataSourceSpecStatus!
}
```
